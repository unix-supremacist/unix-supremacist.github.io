What is this?

this will be a simple no bullshit guide to installing TTW, some basic mods.

this will all be done on linux and assume you have the gog versions of the games with all dlc.

this also will not be a guide to replace, say, the best of times, because i'll assume you are a adept computer user, and again, that you aren't using windows. though windows has very little differences.

before you start the guide, run each of the games once to be sure all files are generated.

Mod Organizer and TTW

Mod Organizer is a tool we use to install mods, using a Virtual File System (aka VFS) to install mods without damaging our games installs, download the 7z from here

check out my linux gaming guide to install and configure wine if you have not (how the fuck did you install f3/fnv on linux without doing that?), then run the mo2 executable.

create a portable instance, it should detect your gog install, select ttw.

if it asks you where to keep the profile, put this in whatever location you want your mods installed to, for example mine is in ~/Games/TTW.

disable FalloutNV_lang.esp on the plugins tab on the right side of MO2.

now go to the puzzle icon at the top of mo2 and open the ini editor, paste the following.


[General]
bUseThreadedAI=1
iNumHWThreads=2
bPreemptivelyUnloadCells=1

[BackgroundLoad]
bBackgroundCellLoads=1
bSelectivePurgeUnusedOnFastTravel=1

[Display]
bFull Screen=1
[Grass]
fGrassStartFadeDistance=7000

[Controls]
fForegroundMouseAccelBase=0
fForegroundMouseAccelTop=0
fForegroundMouseBase=0
fForegroundMouseMult=0

[Audio]
iAudioCacheSize=8192
iMaxSizeForCachedSound=1024

[Archive]
SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa

download ttw 3.2, 3.2.1 and 3.2.2 from here.

extract the 3.2 to a new folder, and then extract the other updates to the same folder.

open a terminal where you have the installer and run, wine TTW\ Install.exe -NoOggEnc2 select the versions in order, and continue into the installer, note, you need to set it to install ttw, and select ttw where you put your MO2 folder, for example i had to bPreemptivelyUnloadCells ~/Games/TTW/mods/TTW/, the installer is self explanatory.

download TTW wrist fix from here install it by the disc button on mo2, then naming it TTW to merge it into your main TTW mod.

download NVMIM from here, install it as before, and also merge this mod into ttw, because it will be included in the next major version of ttw.

once done, activate in mo2, boot the game with mo and test it out, TTW it self should be installed, note some audio might crackle, will be fixed later in the guide.

NVSE Plugins

this section will cover nvse plugins, or in laymen terms, engine mods.

first we need to update the NVSE that came with ttw though, download nvse from here and then overwrite the files in your fallout new vegas installation with these.

download JIP ln from here, and make sure it overwrites TTW on the left pane, don't merge mods from this point on with TTW.

download johnny guitar NVSE from here and install it in the same manor

WARNING THE REST OF THESE NVSE PLUGINS WILL REQUIRE A NEXUS ACCOUNT, I DON'T LIKE IT AS MUCH AS YOU BUT ITS UP TO THESE MOD AUTHORS TO UPLOAD ELSEWHERE.

these mods aren't necessary but will make the gameplay experience smoother and more stable.

you will need stewietweaks to fix audio crackling though.

download nvac from here, using the manual download on nexus from this point on, and install as before.

download nvtf from here, and install as before.

download nvhr from here, and install as before.

download FNV Mod Limit Fix from here, and install as before.

download Infinite Loading Screen Fix from here, and install as before.

download Improved Lighting Shaders from here, and install as before.

download Console Paste Support from here, and install as before.

download Improved Console from here, and install as before.

download OneTweak from here, and install as before, do note the default config will do borderless fullscreen which crashes on Linux, like windowed mode by default, set BorderlessWindow to false if you don't use my configs.

download yUI from here, and install as before.

download yGTM from here, and install as before, note you technically dont have to have this if you install JAM later, but id rather install it separately for updates.

download UIO from here, and install as before.

if you care about discord, download Discord Rich Presence from here, and install as before, this probably doesn't even work on linux.

now to our final plugin, and the one required for most configuration and the audio fix. download lStewieAl's Tweaks from here, and install as before, if you don't plan to download my hardcore configs later, download the ini from the optional files.

Normal Mods

download ySI from here, this will add sorting icons to your inventory, make sure you install yUI first, set bSortingIcons to 1, in the yUI,ini if you don't use my configs.

download MCM and the bugfix update from here, and install both with the same name on mo2.

download DarnUI from here, as well as the MCM matches HUD, note you don't need a nexus account for this one, but you do for MCM to configure it.

download Anniversary Anim Pack from here, as well as the ttw file, and install them under the same name.

download Just Mods Assorted from here, and install as before.

download BLEED from here, and install as before.

download NPC's Sprint in Combat from here, and install as before.

when installing more mods than this, use The Method to ensure no conflicts.
